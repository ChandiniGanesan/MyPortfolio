Data Visualization of Church data for research


Libraries used:Folium,render_template_string, request, jsonify, url_for, HeatMap,MarkerCluster,pandas

Description:
Today we have lot of tools to perform data analysis, however it is essential in some research projects that the researchers have their custom built tools to ease the access to their data and perform statistical analysis using data visualization. In this project, I have buil an web application using HTML and Python to allow users to upload a data.csv file with multiple features of historical Church dataset of the world(In video:Rome). The user can feed in the combination of feature values that needs to be selected for the geospatial visualization. The slider on top of the output indicates (variable) the centuries that appear in the given dataset. Plots that are in green color are the physical location of the churchs that originated in that particular century. The green plot indicated the mathematically calculated center of the churchs that were originated in that particular century.
This code/data is not available for public view.

Implementation of Apriori Algorithm

Libraries used:Javax.swing, BufferedReader, FileReader, Synthetica

Description:
One of the most important and well known data mining concept that helps improve sales in a business is Market Basket Analysis. Apriori Algorithm plays a vital role in affinity analysis that discovers co-occurrence relationships among products. This software developed using Eclipse Java provides users the facility to feed the available purchase history of products and obtain the results of apriori algorithm to understand how purchase of one product affects the other using a support and confidence threshold. Support refers to how many times a combination of products have occurred and confidence is the conditional probability of purchase of product B given purchase of product A.
